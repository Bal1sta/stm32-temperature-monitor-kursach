# STM32 Temperature Monitor

## 📌 Проект включает:

- измерение температуры (датчик TC77, интерфейс SPI)
- отображение температуры на 7-сегментном индикаторе
- светодиодную индикацию состояния
- управление через UART (Terminal)
- команды управления системой (start / stop / reset)

## ⚙️ Команды терминала

- `s` — запуск системы  
- `p` — остановка  
- `r` — перезагрузка  
- `t` — вывод температуры  

## 🛠️ Использовано

- STM32F103  
- Proteus  
- Keil uVision  
- SPI / UART
